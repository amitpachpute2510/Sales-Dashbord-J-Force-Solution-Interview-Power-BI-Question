# 📊 Sales Data Analysis Dashboard | Power BI Project

## 📌 Project Overview

This project was completed as part of a **Data Analyst Assessment**. The objective was to clean and transform sales data using **Python**, store the processed data in **MySQL/Excel**, and build an interactive **Power BI Dashboard** to visualize business insights.

---

## 🎯 Project Objectives

- Import raw sales data into MySQL Database.
- Perform data cleaning and transformation using Python.
- Create a new column **Total Sale Amount = Quantity Sold × Unit Price**.
- Store the cleaned data in MySQL/Excel.
- Build an interactive Power BI dashboard.
- Generate meaningful business insights through visualizations.

---

## 🔄 Project Workflow

### 📥 Data Import
- Imported raw datasets into Python.

### 🧹 Data Cleaning & Transformation
- Removed duplicate records
- Handled missing values
- Corrected inconsistent data
- Converted data types
- Formatted date columns
- Created **Total Sale Amount** column

### 💾 Data Storage
- Exported cleaned data to:
  - MySQL Database
  - Microsoft Excel

### 📈 Dashboard Development
Imported the cleaned dataset into Power BI and created an interactive dashboard.
